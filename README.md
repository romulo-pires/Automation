# API Test Automation with Playwright
Simple API testing project using Python + Playwright + Pytest.

## Features

- API testing using Playwright and Python
- Pytest test structure
- Validates status codes and response body 
- ReqRes used for API calls and responses

## Tech Stack

- Python
- Playwright
- Pytest

## Setup

1. Clone the repo:
    git clone 
    cd 

2. Install dependencies
    pip install -r requirements.txt

3. Install Playwright
    playwright install

4. Run tests:
    py -m pytest -s -v
